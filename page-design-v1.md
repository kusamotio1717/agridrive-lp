> agridrive.jp（STUDIO）+ shop.agridrive.jp（BASE）を1ページに統合

## デザイン方針

- **ブランドページのトーンを完全踏襲**（深緑 #2d4740 / クリーム #f4f1e9 / ライム #b2cf81 / 黄 #fff200）
- フォント：Montserrat（英語）+ Noto Sans JP（日本語）
- 広告からの流入を想定 → ファーストビューで「何の商品か」「いくらか」「何が良いか」を3秒で伝える
- 購入ボタンは複数箇所に配置（スクロールのどこからでも購入に進める）
- 大量ロット＝問い合わせ導線を明確に分離

## 現在のブランドページからの変更点まとめ

### 追加するセクション
1. **HERO にキャッチ + 価格 + 購入CTA** → 広告LP対応の最重要変更
2. **SEC 3: PRODUCT（商品ラインナップ）** → サイズ別の商品カード + 各購入ボタン
3. **SEC 7: 中間CTA** → スクロール途中での購入導線
4. **大量ロット導線** → HERO、PRODUCT、中間CTA、ORDER、CONTACTの5箇所に設置

### 既存セクションの調整
- HEADER → 固定ヘッダーに変更、購入ボタン常時表示
- ABOUT → 短縮してHEROに統合
- TECHNOLOGY → そのまま踏襲
- COST → そのまま踏襲（差し引きメリットの数字を強調）
- STORY → そのまま踏襲
- ORDER → 少量EC購入 / 大量見積もりの2パターン明記
- FAQ → ロット関連FAQ追加
- CONTACT → 問い合わせ種類に「大量ロット見積もり」追加

### 削除・統合するセクション
- 「ECサイトはこちら→」ボタン → 不要（このページ内で完結するため）
- 物性コントロール（Substance Control）→ TECHNOLOGYに統合

## 広告LP としてのポイント

1. **ファーストビュー完結** → 商品名・価格・CTAが見える
2. **購入CTAは5箇所以上** → HERO、PRODUCT各商品、中間CTA、ORDER、固定ヘッダー
3. **大量ロット導線は3箇所以上** → HERO、PRODUCT、中間CTA、CONTACT
4. **ページ内スクロールナビ** → 固定ヘッダーからセクションジャンプ
5. **スマホ最適化** → 商品カードは1列表示、CTAボタンは大きく

## BASE連携の実装方法

各商品の「購入する →」ボタンは以下のいずれかで実装：

**方法A: BASEの商品ページに直リンク**
- `https://shop.agridrive.jp/items/[商品ID]` にリンク
- シンプル。BASEのカート機能をそのまま使える

**方法B: BASE外部カートボタン埋め込み**（推奨）
- STUDIOのカスタムコードブロックにBASEの「カートに入れる」ボタンを埋め込み
- ページ遷移なしでカートに追加できる
- ただしSTUDIOでのカスタムコード制約あり（要検証）

**方法C: 方法Aで開始 → 将来的にBに移行**（現実的）
